# Chocolate Factory Macro
# Features
- **Auto Upgrade Rabbits**: Automatically upgrades your rabbits as soon as upgrades become available
- **Auto Activate Time Tower** 
- **Auto Click Bonus Bunnies**  
- **Eggs ESP**: Highlights eggs through walls using esp	
 ## Common Questions
 **Q: Does Auto Click Bonus Bunnies work even if the game window is not in focus?**  
 **A: Yes the macro is fully afk requiring no human input.**
 
 **Q: What Parts Of The Chocolate Factory Require Human Input**
 **A: The current version of the macro can complete the entire Chocolate Factory from start to finish. The only human input required is for Prestiging.**
