title: "GlaciteCommissions"
description: "Glacite commissions macro "
---

<Danger>
  It's **important** you enable **World Caching** in the [**Pathfinding**](../misc/pathfinding) module, so it works for the longer distances.
</Danger>

<Accordion title="General" icon="gear" iconType="solid">
  - **Commissions Macro Keybind**: toggles Glacite Commissions macro when pressing the keybind
  - **Use Routes**: toggles route usage for glacite commissions macro
  - **Notify on HOTM Level Up**: sends a message on HOTM level up
  - **Disable Macro on Mineshaft Spawn**: toggles auto disable for Glacite Commission macro on Mineshaft Spawn
  - **Notify on Pet Level Up (100 or 200)**: sends a message on Pet Level Up
</Accordion>

<Accordion title="Tunnels Miner" icon="pickaxe" iconType="solid">
  <Info>
    Only use those routes if you have "Waypoint Mode" enabled in [Tunnels Miner](/modules/mining/tunnels-miner)
  </Info>
  - **Glacite Route**
  - **Tungsten Route**
  - **Umber Route**
</Accordion>

<Accordion title="HUD" icon="sparkles">
  - **Reset HUD**: resets HUD
</Accordion>
