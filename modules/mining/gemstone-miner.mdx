<Accordion title="Commands" icon="sparkles">
  - **/n gemstone help**: prints a help message
  - **/n gemstone add [index]**: adds waypoint with selected index
  - **/n gemstone insert [index]**: inserts waypoint with selected index
  - **/n gemstone remove [index]**: removes waypoint with selected index
  - **/n gemstone clear**: clear all waypoints
  - **/n gemstone save [file]**: save waypoints to a file
  - **/n gemstone load [file]**: loads waypoints from a file
  - **/n conver-polar [file]**: converts polar route to nebula format
  - **/n convert-skyskipped [file]**: converts skyskipped route to nebula format
  - **/n convet-melody [file]**: converts melody route to nebula format
</Accordion>
